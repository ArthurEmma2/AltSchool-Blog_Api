# Blog API

A blog API built with Node js

## Features
- ### Authentication
    - Register
    - Login 
    - Logout
    - Stateful Jwt Authentication
    - Create Posts
    - Get Posts
    - Update Posts
    - Delete Posts

## Major Dependencies
- Mongoose - Interfacing with mongodb 
- Express js - The core of the API
- bcrypt - Password hashing
- jwt - generating and verifying tokens
- jest - Testing
